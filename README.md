# IS-IS IPv6 Lab (GNS3)

## Overview
This lab demonstrates IS-IS routing protocol configured 
for IPv6 using Multi-Topology (MT) on GNS3.

## Protocols Used
- IS-IS (Integrated IS-IS)
- IPv6 (Multi-Topology support)
- Dual-stack (IPv4 + IPv6)

## Tools
- GNS3
- Cisco IOS Routers

## How to Open
1. Install GNS3 on your PC
2. Import isis ipv6.gns3
3. Start all devices and verify routing
